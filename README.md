# StudentHub

Projekt akademicki przygotowany w ramach kursu "Programowanie zaawansowanych serwisów internetowych" na uczelni WSB-NLU.

## Status prawny i zasady użycia

Kod w tym repozytorium został udostępniony wyłącznie do wglądu, nauki oraz niekomercyjnych celów edukacyjnych.

Nie wyrażam zgody na:
- komercyjne wykorzystanie kodu,
- kopiowanie całości lub istotnych fragmentów i publikowanie ich jako własnego projektu,
- oddawanie tego projektu lub jego części jako pracy innego studenta,
- usuwanie informacji o autorstwie.

Szczegółowe zasady znajdują się w pliku `LICENSE`.
